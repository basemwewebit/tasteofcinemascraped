# كيف تشتغل على tasteofcinemascraped

> **توثيق شامل:** للتفاصيل الكاملة (المعمارية، API، WebP، السلوق، استكشاف الأخطاء) راجع **[التوثيق_الشامل.md](التوثيق_الشامل.md)**.

## 1. في ووردبريس

1. من **لوحة التحكم** → **الإضافات** → فعّل **Taste of Cinema Scraped Import**.
2. اذهب إلى **الإعدادات** → **TOC Scraped**.
3. ضع **مفتاح سري** (كلمة سر تحددها أنت، مثل: `my-secret-2024`) واحفظ.
4. انسخ نفس المفتاح؛ ستحتاجه في ملف `.env` في الخطوة التالية.

**روابط المقالات (اختياري):** إذا أردت أن يكون رابط المقال مثل  
`http://yoursite.com/10-great-crime-thriller-movies-you-probably-havent-seen/`  
**بدون** ظهور التصنيف في المسار، اختر في **الإعدادات → الروابط الدائمة** خيار **«اسم المقال»** (Post name) ثم احفظ. إن كان الهيكل يحتوي على التصنيف (`%category%`) فسيظهر سلوق التصنيف قبل سلوق المقال.

---

## 2. إعداد البيئة (مرة واحدة)

من **مجلد الإضافة** `tasteofcinemascraped-wp` (حيث يوجد ملف الإضافة وبداخله مجلد `tasteofcinemascraped`):

```bash
cd wp-content/plugins/tasteofcinemascraped-wp

# إنشاء بيئة افتراضية وتثبيت الحزم
python3 -m venv .venv
source .venv/bin/activate    # لينكس/ماك
# أو على ويندوز:  .venv\Scripts\activate

pip install -r tasteofcinemascraped/requirements.txt
```

ثم ضبط ملف الإعدادات:

```bash
cp tasteofcinemascraped/.env.example tasteofcinemascraped/.env
# ثم عدّل tasteofcinemascraped/.env
```

في `.env` تأكد من:

| المتغير | ماذا تضع |
|--------|----------|
| `OPENROUTER_API_KEY` | مفتاحك من [openrouter.ai](https://openrouter.ai) (للترجمة) |
| `OPENROUTER_MODEL` | مثلاً: `openai/gpt-4o-mini` أو `anthropic/claude-3.5-sonnet` |
| `WORDPRESS_URL` | عنوان موقعك، مثل: `http://tasteofcinemaarabi.test` أو `https://yoursite.com` |
| `WORDPRESS_SECRET` | **نفس** المفتاح السري الذي وضعته في الإعدادات → TOC Scraped |

## 3. التشغيل من الطرفية

يجب أن تكون في **مجلد الإضافة** `tasteofcinemascraped-wp` (نفس المجلد الذي فيه `tasteofcinemascraped-wp.php` ومجلد `tasteofcinemascraped`):

```bash
cd wp-content/plugins/tasteofcinemascraped-wp
source .venv/bin/activate   # إن لم تكن المفعلة مسبقاً
```

### تجربة بدون استيراد (سكراب + ترجمة فقط، بدون ووردبريس)

```bash
python -m tasteofcinemascraped "https://www.tasteofcinema.com/2026/10-great-crime-thriller-movies-you-probably-havent-seen/" --dry-run
```

هذا يجرب السكراب والترجمة ويطبع ملخصاً ولا يرسل شيء لووردبريس.

### تشغيل كامل (سكراب → ترجمة → استيراد لووردبريس)

```bash
python -m tasteofcinemascraped "https://www.tasteofcinema.com/2026/10-great-crime-thriller-movies-you-probably-havent-seen/"
```

يُنشأ منشور **مسودة** في ووردبريس مع المحتوى المترجم، الصور، الكاتب، التصنيفات والتاغات. إذا كان نفس الرابط مستورداً مسبقاً، يتم تخطيه. **تاريخ النشر الأصلي** للمقال يُحفَظ. الصور تُحوَّل تلقائياً إلى **WebP مضغوطة** قبل الرفع إلى مكتبة الميديا.

### سكراب سنة كاملة (أو أرشيف)

لمعالجة كل مقالات سنة معيّنة (مثلاً 2026) مع الحفاظ على تاريخ نشر كل مقال:

```bash
python -m tasteofcinemascraped --year 2026
```

أو من رابط أرشيف مخصّص:

```bash
python -m tasteofcinemascraped --archive "https://www.tasteofcinema.com/2026/"
```

- يُكتشَف عدد المقالات من صفحة الأرشيف وتصفحتها (page/2, ...)، ثم يُنفَّذ سكراب → ترجمة → استيراد لكل مقال.
- المقالات المستوردة مسبقاً (نفس الرابط) تُتخطى.
- بين كل مقال وآخر يُطبَّق تأخير افتراضي 2 ثانية (تعديله: `--delay 1.5`).
- تجربة بدون استيراد: أضف `--dry-run`.

### استيراد بدون ترجمة (محتوى إنجليزي كما هو)

```bash
python -m tasteofcinemascraped "https://..." --no-translate
```

## ملخص سريع

| ماذا تريد | الأمر |
|-----------|--------|
| تجربة بدون ووردبريس | `python -m tasteofcinemascraped "الرابط" --dry-run` |
| استيراد كامل (عربي) | `python -m tasteofcinemascraped "الرابط"` |
| استيراد إنجليزي فقط | `python -m tasteofcinemascraped "الرابط" --no-translate` |
| سكراب سنة كاملة (مع الحفاظ على تاريخ النشر) | `python -m tasteofcinemascraped --year 2026` |
| سكراب من رابط أرشيف | `python -m tasteofcinemascraped --archive "https://www.tasteofcinema.com/2026/"` |

**مهم:** شغّل الأوامر من داخل مجلد الإضافة `tasteofcinemascraped-wp` بعد تفعيل الـ venv.
