# Getting Started: TasteOfCinema Scraper WP Admin

1. Verify `.venv` exists in your plugin directory and dependencies from `requirements.txt` are installed (this is a prerequisite that admins must handle from their terminal first, as per fail-fast requirements).
2. Go to the WP Admin dashboard.
3. In the left menu, locate "Taste Of Cinema Settings" -> "Scraper"
4. To scrape a single article, paste the URL in the single input field and hit "Run". Watch the progress log box.
5. To import a batch, pick the Year (and optionally) the Month. Hit "Import Batch". Watch the real-time progress.
